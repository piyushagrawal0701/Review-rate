# 📌 Review Rate – MERN Stack Company Review System

A full-stack web application where users can browse companies, add reviews, and view ratings. Built using the MERN stack (MongoDB, Express, React, Node.js) with Cloudinary for image uploads.

---

## 🚀 Live Links

- 🌐 Frontend: https://reviewrate.netlify.app/
- ⚙️ Backend API: https://review-rate-eosin.vercel.app/

---

## 🧠 Features

- User authentication (JWT-based)
- Add / view companies
- Add reviews and ratings
- Auto-calculated average rating
- Company logo upload using Cloudinary
- Search & filter companies
- Fully responsive UI

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Axios
- Tailwind CSS
- React Router

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- Cloudinary (image upload)
- Multer (file handling)
- JWT Authentication
